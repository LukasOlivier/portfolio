Tappd - Online ordering service Personal

Tappd is an online ordering service for bars and restaurants. It allows customers to order their drinks and food without having to wait for a waiter.

The waiter receives the order on a tablet and can then serve the customer.
The owner can manage the menu and available tables.

The client is made using vanilla HTML, CSS & JavaScript and the server is made using Laravel.
